Projects

Cloudify

A Spotify Clone created using React-Redux, Javascript, and Ruby on Rails as the backend. Allows users to play, pause, skip, and search for music, as well as creating their own playlist in which they can rename, delete, or add songs to!

Running Idle

Leading a group of like-minded individuals, created a workout app turned game using the MERN stack, implementing google maps to track user movement. Gamified physical activity, rewarding users based on distance traveled. Displays runs in a leaderboard format!

Record Player

A Fun, enjoyable interactive music experience that uses a music api to dynamically alter the music being played, while allowing users to change up and enjoy a visual experience with their music! Built completely in vanilla JavaScript.
